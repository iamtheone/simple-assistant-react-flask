# Building a Modern AI Chat Application

Welcome to our comprehensive course on building a modern AI chat application! This project-based course will guide you through creating a sophisticated chat application using Flask, React, and OpenAI's Assistants API.

## Course Structure

### Part 1: Foundation and Setup
- [Lesson 1: Setting Up Flask and React](/lessons/1-foundation)
  - Basic project structure
  - Flask backend setup
  - React frontend initialization
  - Basic client-server communication

- [Lesson 1A: OpenAI Assistants Integration](/lessons/1a-openai-assistants)
  - Understanding the Assistants API
  - Thread management
  - File handling
  - Advanced AI features

### Part 2: Enhanced Features
- [Lesson 2: Modern UI with ShadCN](/lessons/2-enhanced-ui)
  - Setting up ShadCN UI
  - Creating custom components
  - Implementing chat features
  - Error handling and loading states

## Prerequisites
- Basic knowledge of Python and JavaScript
- Familiarity with React concepts
- OpenAI API key
- Node.js and npm installed
- Python 3.8 or higher

## What You'll Build
By the end of this course, you'll have created a professional-grade chat application featuring:
- Modern, responsive UI
- Real-time message streaming
- AI-powered responses
- File upload and processing
- Persistent chat history
- Error handling and loading states

## Getting Started
Begin with [Lesson 1](/lessons/1-foundation) to set up your development environment and create the basic application structure. Each lesson builds upon the previous one, so it's recommended to follow them in order.

## Source Code
The complete source code for this project is available in the `demo-project` directory. Feel free to use it as a reference as you work through the lessons.
